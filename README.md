# AI伴学与职业成长平台 (AI Companion Platform)

本项目是一个面向高校学生的智能学习辅助系统，旨在提供基于大语言模型的学业辅导、技能树管理与模拟面试服务。

## 项目架构

项目采用经典的前后端分离与移动端多端协同架构：
- **后端服务 (Backend)**: Spring Boot 3 + MyBatis-Plus + MySQL 8.0，提供 RESTful API，并集成 Dify/DeepSeek 大语言模型接口。
- **管理后台 (Web Frontend)**: Vue 3 + Vite + Element Plus + Pinia，提供给管理员进行用户管理、数据看板、技能树维护和学习记录审查。
- **移动端 (HarmonyOS)**: 基于 HarmonyOS NEXT (ArkTS/ArkUI) 开发的移动端 App，提供给学生用户进行 AI 对话、技能点亮和模拟面试。

## 核心功能模块

### 1. 移动端 (学生端)
- **技能树追踪**：可视化展示专业技能树，点击卡片标记“学习中”或“已掌握”。
- **AI 对话辅导**：集成 Dify 智能聊天流，解答专业技术问题。
- **模拟面试沙箱**：针对所选岗位生成专业面试题，AI 自动评分并输出优化建议。
- **个人数据看板**：实时汇总累计学习时长与已掌握技能项数。

### 2. Web 管理端 (管理员端)
- **全局数据总览**：ECharts 图表展现注册用户量、各分类技能热度与每日 AI 调用趋势。
- **用户权限管控**：浏览注册用户列表，支持账号启停与角色提权。
- **技能体系维护**：通过树状控件可视化管理全局的技能体系。
- **学习履历审计**：浏览各用户的学习笔记记录（当学生在移动端点亮技能时，后台会自动生成对应日志）。

## 开发环境要求

- **Java**: JDK 17+
- **Node.js**: v18+ (用于 Web 前端)
- **IDE**: IntelliJ IDEA (后端) / DevEco Studio (移动端) / VS Code (Web前端)
- **数据库**: MySQL 8.0+
